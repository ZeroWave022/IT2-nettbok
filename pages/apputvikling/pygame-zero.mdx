# Pygame Zero

Pygame zero er et bibliotek som er bygd oppå Pygame.
Den fjerner mye av koden som er nødvendig for å sette opp et spill i Pygame.

## Ressurser

- [Uio: komme i gang med pygame zero](https://www.uio.no/studier/emner/matnat/ifi/IN1000/h21/undervisningstilbud/alternativt-oblig-lop/week-8/pygame-zero.md)

- [Pygame Zero: examples](https://pygame-zero.readthedocs.io/en/stable/examples.html)

## Installering

- mac: `python3 -m pip install pgzero`
- windos: `python -m pip install pgzero`
